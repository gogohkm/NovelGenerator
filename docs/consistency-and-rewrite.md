# 모순 감지와 영향 재생성 규칙

## 모순 감지

`ConsistencyChecker`는 다음 검사를 제공합니다.

- 사실 충돌 검사 (`check_fact_conflicts`)
  - 동일 사실 키에 상이한 값이 등록되면 `high` 이슈 생성
- 세계관 규칙 위반 검사 (`check_world_rule_violations`)
  - 장면 텍스트/요약에서 규칙 위반 마커가 발견되면 이슈 생성

## 사실 업데이트 정책

`update_fact(...)` 규칙:
- 기존 사실이 `IMMUTABLE`인데 값 변경 시도 시 예외 발생
- 그 외는 최신 장면 기준으로 사실 갱신

## 분기 변경 영향 분석

`RewriteImpactAnalyzer.analyze(...)`는 분기 수정 시 재생성 범위를 계산:

- 직접 영향: 변경된 분기에 속한 장면
- 간접 영향: 하위 분기(자손 브랜치)의 장면
- 우선순위:
  - `1`: 직접 영향 장면 (즉시 재생성)
  - `2`: 초중반 간접 영향 장면
  - `3`: 후반 간접 영향 장면

이 결과를 활용하면 분기 수정 후 전체를 다시 쓰지 않고 필요한 범위부터 순차 리라이트 가능.
