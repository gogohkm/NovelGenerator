# MVP 범위 및 지표

## MVP 범위

- 프로젝트 설정: 장르/톤/시점/세계관 규칙
- 4막 트리 기본 구조
- 분기 선택지 3개 생성 + 사용자 선택 반영
- 장면 카드 생성 및 저장
- 계층형 메모리(전역 사실, 막 요약, 최근 장면)
- 기초 모순 감지(사실 충돌, 세계관 위반 마커)
- 분기 변경 영향 분석(직접/간접/우선순위)

## 제외 범위 (MVP 이후)

- GUI 완성형 에디터
- 실제 상용 LLM 멀티모델 오케스트레이션
- 고급 문체 학습/강화
- 협업/권한/클라우드 동기화

## 성과 지표 (초기)

- 완주율: 30장면 파일럿에서 중도 붕괴 없이 마지막 장면까지 도달 비율
- 모순률: 장면 100개당 `high` 정합성 이슈 개수
- 사용자 체감 품질: 선택-결과 연계 만족도(5점 척도)

## 측정 운영안

- 파일럿 20~30장면 프로젝트 5개 생성
- 매 5장면마다 정합성 검사 자동 실행
- 분기 수정 발생 시 영향 분석 결과와 실제 수정 범위 비교 기록
